# India-Power-Grid-Demand-Forecasting
⚡ Built an end-to-end ML pipeline for electricity demand forecasting

📊 Achieved high accuracy (R² ≈ 0.95) with low error metrics (RMSE, MAE, MAPE)

🧠 Applied Ridge Regression (RidgeCV) with automated hyperparameter tuning

🕒 Performed advanced feature engineering using temporal & lag features

📈 Implemented time-series forecasting (24-hour & multi-day predictions)

💻 Developed an interactive Streamlit dashboard for real-time insights

🔍 Enabled data exploration across hourly, monthly, and yearly trends

🚨 Integrated grid load classification (Low / Normal / High / Critical)

💡 Generated actionable insights & smart recommendations for demand management

🎨 Designed a modern, visually appealing UI (cyberpunk theme)
